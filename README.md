# WALKRI: Working Architecture for Legible, Knowledge-Ready Intake

Version 0.1.0 | 2026-05-15 | CC0

WALKRI is a companion standard to CROSS (Common Reporting Outcome Standards Schema). Together they are branded CROSS+WALKRI.

CROSS specifies what funded interventions must demonstrate. WALKRI specifies how the fields and criteria through which that demonstration is collected must be defined, so the resulting data is valid, consistent, provenance-aware, and reusable by both humans and machines.

WALKRI is portable: it works with any obligation standard, not only CROSS.
